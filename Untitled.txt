The system must look extremely modern, luxurious, clean, responsive, smooth, and production-ready. The UI/UX should feel comparable to premium food delivery platforms and SaaS admin dashboards.

* Mobile responsiveness

* Advanced admin controls
* Secure architecture
* Scalable database design

* Next.js latest version
* TypeScript
* TailwindCSS
* Framer Motion
* ShadCN UI
* Supabase for backend/database/auth/storage
* Chart.js or Recharts for analytics

* React Query / TanStack Query
* Fully responsive mobile-first design
* Dark/Light mode
* Modern glassmorphism + premium dashboard design
* Clean typography
* Smooth page transitions
* Loading skeletons
* Toast notifications
* Optimized SEO

The design should feel like:

* Foodpanda
